# 🚀 TechHub - Modern Business Website

A modern, responsive, and professional multipage-style business website built using **HTML, CSS, and JavaScript**. The website showcases services, featured products fetched from an API, a contact form, and a clean user-friendly interface following modern UI/UX principles.

---

## 📖 Overview

TechHub is designed as a modern business landing page that helps companies showcase their services and products in a visually appealing way. The project demonstrates responsive web design, API integration, and clean frontend development practices.

---

## ✨ Features

- 🎨 Modern and professional UI design
- 📱 Fully responsive layout for all devices
- 🧭 Sticky navigation bar
- 🦸 Hero section with call-to-action buttons
- 🛠️ Services section with feature cards
- 🛍️ Dynamic product cards loaded from an API
- 📬 Contact form section
- 🚀 Smooth scrolling navigation
- ⚡ Fast and lightweight implementation
- 🎯 Mobile-first responsive design
